# Usage
<!-- usage -->
```sh-session
$ npm install -g sonoff-diy
$ sonoff-diy COMMAND
running command...
$ sonoff-diy (-v|--version|version)
sonoff-diy/0.0.2 darwin-x64 node-v15.7.0
$ sonoff-diy --help [COMMAND]
USAGE
  $ sonoff-diy COMMAND
...
```
